# Landing Page

A modern responsive landing page for a startup-investor platform built using HTML, CSS, and JavaScript.

## 🚀 Features

- Responsive design (mobile + desktop)
- Modern UI with hover animations
- Sticky navigation bar
- Hero section with CTA buttons
- Startup categories
- Startup listing cards
- Investor section
- Clean UI with shadows and hover effects

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
project-folder/
│
├── index.html
├── style.css
├── script.js
└── README.md

## ⚙️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

## 📱 Responsive Design

The website works on:
- Desktop
- Mobile devices

## 🎯 Purpose

This project was created as part of a frontend assignment to demonstrate skills in building a responsive and modern landing page UI.
